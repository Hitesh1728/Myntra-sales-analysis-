Myntra Sales Analysis Dashboard
📌 Project Overview

This project presents a comprehensive analysis of Myntra’s sales data using Power BI and Excel. It focuses on uncovering key insights related to customer behavior, product performance, and revenue trends to support data-driven business decisions.

📊 Dataset Description

The dataset includes multiple dimensions of sales data such as:

Product details (category, sub-category, brand)
Customer information (location, demographics)
Transactional data (orders, revenue, discounts)
Date and time attributes for trend analysis
🛠️ Tools & Technologies
Power BI – Dashboard creation & data visualization
Microsoft Excel – Data cleaning & preprocessing
DAX (Data Analysis Expressions) – KPI calculations
⚙️ Project Workflow
Data Cleaning & Preprocessing using Excel
Data Modeling (creating relationships between tables)
KPI Creation using DAX (Total Sales, Orders, Avg Sales, etc.)
Dashboard Development with interactive visualizations
📈 Key Features
📊 10+ KPIs including Total Sales, Orders, Avg Discount, Revenue
🔍 Interactive filters (Category, Brand, Year, Sub-category)
📉 Trend analysis (daily, monthly, yearly sales)
🌍 Region-wise sales performance
🏆 Identification of top-performing products and brands
📊 Dashboard Visualizations
Bar & Column Charts – Sales by category and brand
Line & Area Charts – Sales trends over time
Donut Chart – Product distribution
Table View – Detailed transactional insights
Ribbon Chart – Year-over-year comparison
📌 Key Insights
Identified top-performing categories and brands contributing to revenue
Analyzed impact of discounts on sales performance
Discovered regional sales distribution patterns
Highlighted seasonal trends in customer purchasing behavior
🚀 How to Use
Download the .pbix file from this repository
Open it in Power BI Desktop
Explore the dashboard using filters and interactive visuals
